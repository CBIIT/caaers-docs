Adverse Events Module > Enter Expedited Report

Section #10: Conmeds

Document concomitant medications and dates of their administration that are temporally related to the investigational intervention and the event. Consider any medications that might have contributed to an event such as abnormal liver function tests. Report any non-protocol medications that might have contributed to the event.  Medications must be entered one at a time. Each medication entered will appear in the Attribution section of caAERS. The relationship of each medication to each adverse event entered must be addressed.

Click Add a medication to add a Concomitant Medication.  If you want to remove a Concomitant Medication select the x in the right hand corner of the field.

♦    Information about concomitant medication

o  Provide information about the medication the participant was taking outside of the medications being used according to the protocol guidelines.

 

Click Continue when the Conmeds tab is entered completely or if there are no concomitant medications.  The data will be saved to the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
