List Rules Overview

Rules are needed to determine what course of action the system should take when an Adverse Event is logged in the system.  The caAERS application is dependent on the rules set up by the administrator.  Rules will define when an expedited report must be submitted or if the user can continue entering information in a routine report.  This section will show you all the rules created in the application in a table of information.

In the table you can take the following actions:

♦    Enable- This will turn the rule on

♦    Disable - This will turn the rule off

♦    Export/Download - This will generate the rules in an XML file format so it can be imported into another instance of caAERS installed in a different location

♦    Delete - This will remove the rule completely from the application and not allow you to Enable, Disable, or Export/Download it.
