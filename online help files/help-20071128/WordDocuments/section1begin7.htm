Study Module > Create Study

Section #1: Begin

Enter the basic details for the study you want to create in the caAERS application.    

 

Basic Details

♦    Short Title

o  Enter the protocol title intended to be viewed by the public.

♦    Long Title

o  Enter the official name of the protocol provided by the study principal investigator or sponsor.

♦    Precis

o  Enter Official name of the protocol provided by the study principal investigator or sponsor.

♦    Description

o  Enter extended description of the protocol, including information not already contained in other fields, such as comparison(s) studied.

♦    Phase

o  Select the phase of investigation, as defined by the US FDA for trials involving investigational new drugs.

♦    Status

o  Select the status of the study from a drop down box containing the following values; Active, Administratively Complete, Approved, Closed to Accrual and Treatment, Closed to Accrual, Temporarily Closed to Accrual and Treatment, Temporarily Closed to Accrual..

♦    Multi-Institutional

o  This describes the study as occurring across many sites, or just one single site.

Adverse Event Coding Terminology

♦    Terminology

o  Two different types of terminology can be selected, CTC or MedDRA.  CTC terminology comes loaded with caAERS but MedDRA can be imported.

♦    Version

o  Select the version of MedDRA or CTC the study will be using.

Coordinating Center Details

♦    Coordinating Center

o  Enter the name of the coordinating center.  It must be an organization already entered into caAERS.

♦    Coordinating Center Study Identifier

o  Enter the identifier for the coordinating center.

Funding Sponsor Details

♦    Funding Sponsor

o  Enter the name of the funding sponsor.  It must be an organization already entered into caAERS.

♦    Funding Sponsor Study Identifier

o  Enter the identifier for the funding sponsor.

 

 

Click Continue when the Basic Details section is completed.  If any data fields are entered incorrectly, an error message will appear after Continue is clicked.  Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  This will save the data in the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
