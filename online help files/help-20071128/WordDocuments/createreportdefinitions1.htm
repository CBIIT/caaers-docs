Create Report Definitions

Report Definitions are needed to determine who, what, and when will receive a notification that an AE has occurred in a study.  A report definition will contain the people who should be notified, when they should be notified, and how they should be notified.  The definition will also determine what fields are mandatory for a user to enter during an expedited report submission.

The process will take you through 5 screens before the Report Definition will be saved.  Once it is created it can be used in a rule set that is created in the Manage Rules section.

Every field in the online help that references a required field means it is required in all circumstances.  Other optional fields will also be explained.
