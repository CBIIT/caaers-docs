Study Module

Create Study

The caAERS application allows a user to create studies via a web based interface.  Once a study is created, participants can begin to be assigned to it so adverse events can begin to be tracked.  There are 10 screens of information that can be captured before the study is created, but not every screen will have required information.

Every field in the online help that references a required field means it is required in all circumstances.  Other optional fields will also be explained.
