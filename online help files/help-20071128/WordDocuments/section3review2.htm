Section #3: Review

caAERS provides a final page in the participant creation to review all of the information supplied in the previous pages.  If every field is accurate, you may Save the information in the database.

Once you save the data, a confirmation page will be displayed to show the participant is in the database and ready for use.

 

Click Save when the Review tab is reviewed completely.  The data will be saved to the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
