Rules Module > Manage Rules

Section #1: Rule Level

There are four options provided to you to select from, Sponsor Rules, Institution Rules, Sponsor defined rules for the study and Institution defined rules for the study.  The first two will affect every study being run by either Sponsor or the Institution.  The last two are going to affect only a specific study at the particular Sponsor or Institution.  

 

Rule Level

♦    Select the radio button next to the level you want to apply this new rule to.

o  Selecting either Sponsor or Institution will display a single text field to enter either the Sponsor or Institution.

o  Selecting Sponsor defined or Institution defined will display two text fields.  One will be to enter the sponsor or institution and the other one is for the study.

 

The text fields will populate with a list of options to choose from as you begin entering information.

 

Click Continue when the Rule Level section is completed.  If any data fields are entered incorrectly, an error message will appear after Continue is clicked.  Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  This will save the data in the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
