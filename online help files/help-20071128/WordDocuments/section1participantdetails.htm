Create Participants

Section #1: Participant Details

The details for the participant, like Name, DOB, and Gender, must be supplied as well as the Site information.  Once the information is provided you will be allowed to move on to the next section of the process.   

 

Site

♦    Select the site name from the drop down box.  The sites listed will have already been added to the application but if you do not see the one you are looking for you will need to ask your Administrator to add the site to the application for use..

Participant Details

♦    First Name

o  The first name of the participant is a required field

♦    Last Name

o  The last name of the participant is a required field.

♦    Maiden Name

o  The maiden name for the participant is only appropriate for females and is an optional field

♦    Middle Name

o  The middle name of the participant is an optional field

♦    Date of Birth

o  The date of birth of the participant is a required field and must be entered in the DD/MM/YYYY format.

♦    Gender

o  The gender of the participant is a required field.

♦    Ethnicity

o  The ethnicity of the participant is a required field.

♦    Race

o  The race of the participant is a required field.

 

Once the required fields have been entered you will need to enter provide the identifiers assigned to the participants   Select the Add Organization Identifier or Add System Identifier button.

 

Study ID’s

♦    Identifier

o  This is the alpha-numeric identifier for the participant and it is a required field

♦    Identifier Type

o  This can be either a medical record number or other and it is required.

♦    Organization/System Name

o  If you are adding an organizational identifier, this will be a text box that you can enter the name of the organization.  A list will populate based on your entry to choose from.

o  If you are adding a system identifier, this will be a text box to enter the name of the system that generated the identifier

♦    Primary Indicator

o  One of the IDs must be the primary indicator for the participant so select the check box next to the appropriate one.

 

Click Continue when the Participant Details section is completed.  If any data fields are entered incorrectly, an error message will appear after Continue is clicked.  Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  This will save the data in the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
