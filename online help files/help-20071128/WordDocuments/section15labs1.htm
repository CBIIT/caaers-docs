Adverse Events Module > Enter Expedited Report

Section #15: Labs

Click “Add a lab” to add a Lab Result

 

♦    Lab test name

o  Select a descriptive laboratory test name.  If you can not find a lab test name, enter it in the Other field.

♦    Units

o  Enter the unit of measure associated with the lab results. These are mandatory for any Lab selected

 

Values and Dates are mandatory for any Lab. Enter dates (MM/DD/YYYY) by typing in the correct format or by using the calendar.

 

♦    Baseline value/Date

o  Enter the baseline laboratory result and date.

♦    Worst value/Date

o  Enter the worst laboratory result and date.

♦    Recovery value/Date

o  Enter the recovery value of the laboratory and date

 

Occasionally, “Baseline,” “Worst,” and “Recovery” might be the same.  For example, a baseline value may not be available for an abnormal lab that occurred during treatment. In this case, the baseline value and date entered would be the first lab obtained.

Summary of possible date combinations:

♦    Baseline is equal to or less than Worst and/or Recovery

♦    Worst is equal to or greater than Baseline; is equal to or less than Recovery

♦    Recovery is equal to or greater than Baseline and/or Worst

 

Click Continue when the Labs tab is entered completely or if there are no lab information to be supplied.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
