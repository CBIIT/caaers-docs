Enter Expedited Reports

Section #14: Device

Enter any required information for a medical device that was/is being used in this particular study.  The medical device information can be found on the actual device and in any documents that came with the device.

 

Additional information like the operator of the device and whether or not it was implanted in the participant will need to be supplied as well.  Please fill in all of the required fields.

 

Click Continue when the Device tab is entered completely or if there are no devices being used as part of the study.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
