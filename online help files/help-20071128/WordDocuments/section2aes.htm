Manage Reports

Section #2: AEs

A list of all the adverse events reported for the participant and study you selected will be displayed.  Routine and Serious AEs will be displayed on the same page.

 

Expedited Reports:

♦    Term – Click on the term to go to that expedited report and update the information

♦    Actions – You can take the following actions described below

o  Notify PSC – Selecting this link will notify the Patient Study Calendar of this serious adverse event for the participant.  This will only work if your administrator has set up the PSC and provided the correct location for it in caAERS.

o  Generate PDF – Selecting this link will generate the complete report into a downloadable PDF file.

o  Generate XML – Selecting this link will generate the complete report into a downloadable XML file.

 

Routine AEs

♦    Term – Click on the term to go to that expedited report and update the information

 

From the managed reports screen you can also begin entering an Expedited Report or Document a Routine AE.  Select the Create link next to each title to start the new report.  Please go to the online help sections for Enter an Expedited Report or Document Routine AEs for assistance.
