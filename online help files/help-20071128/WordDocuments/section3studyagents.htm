Section #3: Study Agents

Study agents can be added to a study.  The section will appear blank until the Add Study Agent button is selected in the lower right.  This will display a form for you to enter in the appropriate information for the agent. 

 

More than one agent can be added by continuing to select the Add Study Agent button, and if you want to remove a treatment assignment select the red X button.

 

Study Agent

♦    Agent

o  Enter the agent name or identifier in the text field.  Once you enter a value a list will automatically populate with agents you can choose from.

♦    Other

o  If you can not find the agent you are looking for in the previous field, select the radio button next to this field name.  Type the name of the agent in the text field.

♦    Enter IND information – Choose from the following values in the drop down box.

o  NA – Not applicable

o  NA/-Commercial Agent – The agent is a commercial agent that has already been approved.

o  IND-Exempt – The agent is exempt from being classified as an investigational new drug

o  DCP IND – Division of Cancer Prevention holds the IND ownership

o  Other IND Holder – A different organization holds the IND ownership

o  CTEP IND – Cancer Therapy Evaluation Program holds the IND ownership

 

 

 

Click Continue when the Study Agents section is entered completely. If any data fields are entered incorrectly, an appropriate error message will appear after Continue is clicked. Error messages will describe data problems in specific fields. Correct the problems, and click Continue again.  The data will then be saved to the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
