Section #13: Surgery

 

If applicable, supply the surgical intervention that was necessary as a result of this AE by selecting the “Add a surgery” button.  If there was no surgery involved, move on to the next section.

 

Treatment arm and Treatment arm description are going to be displayed based on previous information provided in the expedited report.

 

♦    Intervention site

o  In the text field, begin entering the first few characters of where the surgery site was.  A list will auto-populate with matching criteria.  Select the appropriate site.

♦    Date of intervention

o  Enter the date the participant underwent surgery

 

Click Continue when the Surgery tab is entered completely or if there was no Surgery administered.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
