Search/Edit Participants

Section #1: Search

You can search for a participant by supplying information in 3 different fields.  Not all the fields will need to be populated in order for the search to begin. Searching only requires one character be entered before the application allows you to search.

 

Participant Criteria

♦    Enter information in the following fields, Identifier, First Name and/or Last Name depending on what criteria you would like to search on.  Select the “Search” button after the values have been entered.

Participant Search Results

♦    The application will return any matching participants based on the search criteria provided in the first section of the page.  The matching studies will be in a table broken down in the following way:

o  First Name

o  Last Name

o  Identifier

 

Select the participant you would like to view or edit by clicking checkbox next to the first name.  Once in the overview you can move on to edit information for the study.
