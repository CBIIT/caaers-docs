Adverse Events Module > Enter Expedited Report

Section #18: Submit

A table will be displayed with all of the reports that will be created.  The table will display the Report name, Report ID, Report version and whether or not it is ready for submission.  If a report is not ready for submission, the table will tell you what sections need to be completed before it can be submitted.  Additional information in the table includes Report Status, which will display day the report is due on, and available Actions.

If a report is ready to be submitted, you may select the Submit link under the Actions column.  Before the report is even submitted, it could also be withdrawn by selecting the Withdraw link.  This acts in a way to cancel the report from being submitted.  If you will be submitting the report at another time, select the Save button.  After selecting Save, a list of all the AEs reported for the participant will be displayed.
