Document Routine AEs Overview

The caAERS application allows a user to document routine adverse events via a web based interface.  If you are working with a study using the MedDRA coding vocabulary you will only need to provide information on 3 screens.  If the study you are assigned to is using the CTC coding vocabulary there will be 4 screens to navigate through.

While you are documenting a routine AE, the system might force you to enter an expedited report.  For information on what to do with an expedited report, please click here.

Efficient completion of the routine AE  report is accomplished by working on the sections in order as they appear on the menu.

Depending on the study, different fields could be mandatory or optional.  Every field in the online help that references a required field means it is required in all circumstances.  A field could be mandatory for you as you go through the pages, but it might not be mandatory for another user in another study.  Information for the field will be provided.
