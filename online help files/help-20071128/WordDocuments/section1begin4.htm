Adverse Events Module > Enter Expedited Report

Section #1: Begin

The process of entering an expedited report begins with the selection of a participant and a study. 

 

Select Participant:

♦    Registered study participant entered into the caAERS database by import, API or manually.  This is a required field

♦    In the text field enter the characters incorporated in the participant name or unique identifier.  If you selected study first, only the participants assigned to the study will display.  This is a required field.

♦    The auto-fill text field generates a drop-down list of participants in the system. Select the one you are looking for.

 

Select Study:

♦    Protocol entered into the caAERS database by import, API or manually.  This is a required field

♦    In the text field enter the characters incorporated in the study’s short title or unique identifier. If you selected participant first, only the studies the participant is assigned to will display.  This is a required field.

♦    The auto-fill text field generates a drop-down list of studies in the system.  Select the one you are looking for.

 

Click Continue when the Participant and Study fields are completely filled.  If a data item is entered incorrectly, an error message will appear after Continue is selected.  Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.
