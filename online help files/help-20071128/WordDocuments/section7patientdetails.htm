Section #7: Patient Details

 

Patient Details

Birth date, Race, Ethnicity and Gender will all be supplied based on information entered previously for the participant.  Additional information about the participant can be supplied in the rest of the fields.

♦    Height

o  Enter the value for height in the text field and then select the unit of measure from the drop-down to the right of the field.

♦    Weight

o  Enter the value for the weight in the text field and then select the unit of measure from the drop-down to the right of the field.

Disease Information

♦    Disease name

o  A list of disease names associated with the study will be provided in the drop-down box.  Select the appropriate name.

♦    Other (disease)

o  If the disease name does not appear in the drop-down box, enter the name of the disease in the text field.

♦    Primary site of disease

o  Enter the first few characters of the primary site of the disease in the text field.  A list of appropriate options will auto-fill in the field and allow you to choose the correct one.

♦    Other (site of primary disease)

o  If the primary site of the disease does not appear in the drop-down box, enter it in the text box.

♦    Date of initial diagnosis

o  If known, enter the date of the first diagnosis the patient received for the disease.

 

 

Click Continue when the Patient Details tab is entered completely. If any data fields are entered incorrectly, an error message will appear after Continue is clicked. Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  The data will be saved to the database

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
