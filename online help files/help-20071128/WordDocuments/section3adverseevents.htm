Document Routine AEs

Section #3: Adverse Events

Information must be provided for the routine AE(s) that have occurred.  The steps below will detail what needs to be done for a study using CTC Terms and what need to be done for a study using MedDRA terms.

 

Study using CTC 

♦    Adverse Events

o  Select one of the categories you chose in the previous screen.  A list will populate with all of the CTC terms available for you to choose from.  Choose one of the terms.

o  Select the Grade of the AE from the drop down box of the following values: 0,1,2,3,4,5.

o  Select the Attribution of the AE to the study itself from the drop down box of the following values: Unrelated, Unlikely, Possible, Probable or Definite.

o  Select the Hospitalization as a result of the AE from the drop down box of the following values: None, Hospitalization, Prolonged Hospitalization

o  Select the Expectedness of this AE to occur as a result of the study from the drop down box of the following values: Yes, No

If you need to document more than one AE in this report select the Add button to add another one.  Follow the steps above again.

 

Study using MedDRA

♦    Periods of Observation

o  Provide the start date and end date for the duration of the AE endured by the participant.  Enter the information in the text field or select the dates from the calendar widget.

♦    Adverse Events

o  Enter information in the Terms field and a list will populate with the MedDRA terms available for you to choose from.

o  Select the Grade of the AE from the drop down box of the following values: 0,1,2,3,4,5.

o  Select the Attribution of the AE to the study itself from the drop down box of the following values: Unrelated, Unlikely, Possible, Probable or Definite.

o  Select the Hospitalization as a result of the AE from the drop down box of the following values: None, Hospitalization, Prolonged Hospitalization

o  Select the Expectedness of this AE to occur as a result of the study from the drop down box of the following values: Yes, No

If you need to document more than one AE in this report select the Add another AE button to add another one.  Follow the steps above again.

 

 

Click Continue when the Adverse Event section is entered completely. If any data fields are entered incorrectly, an appropriate error message will appear after Continue is clicked. Error messages will describe data problems in specific fields. Correct the problems, and click Continue again.  The data will then be saved to the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
