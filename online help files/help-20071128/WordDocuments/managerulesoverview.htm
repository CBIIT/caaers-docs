Manage Rules Overview

Rules are needed to determine what course of action the system should take when an Adverse Event is logged in the system.  The caAERS application is dependent on the rules set up by the administrator.  Rules will define when an expedited report must be submitted or if the user can continue entering information in a routine report. 

Rules can be created at four different levels, the Sponsor level, the Institution level, the Sponsor Defined Study level, or the Institution Defined Study level.

Every field in the online help that references a required field means it is required in all circumstances.  Other optional fields will also be explained.
