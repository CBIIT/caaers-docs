Section #4: Notifications

Notifications can be set up in caAERS to remind users they have an un-submitted or unfinished report that is due in a certain timeframe.  The notifications will be sent using the same time scale UOM selected for the report itself.  So, if the user should finish the report in 30 minutes, the notifications can be sent every 5 minutes or 10 minutes depending on what you choose.

 

Select the day/minute/hour etc you want the notification to be sent, and then select the Add Notification button.  The recipients, roles, and URLs added in the previous section will appear.  The subject line in the notification can be entered manually or you can select from a predefined list in the drop down box called Insert a Substitution Variable.

 

 

Click Continue when the Notifications section is completed.  If any data fields are entered incorrectly, an error message will appear after Continue is clicked.  Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  This will save the data in the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
