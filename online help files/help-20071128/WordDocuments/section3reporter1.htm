Section #3: Reporter

 

Reporter Details:

Enter the first name, last name, middle name, email, fax and phone information for the individual entering data into this report.  If an existing study personnel ID is selected in the drop menu for Research Staff, the information will automatically populate these fields. 

Accurate e-mail address for the Reporter is critical because when this screen is saved, an e-mail message will immediately be sent to the Reporter documenting Ticket Number, CTEP Protocol Number, and Patient ID - information required for accessing the report in the future.

 

Physician Details:

Provide information regarding the physician responsible for addressing questions about this report. If the “Reporter Information” represents the physician filling out the report (the physician responsible for addressing questions about this report) click in the box next to “If the Physician is the reporter then check this box.” and the Reporter Information will populate the Physician Information fields.

Because of the medical importance of serious and/or unexpected Adverse Events, the responsible physician should review any Expedited Report prior to submission to CTEP.  Timely reporting of serious Adverse Events is required by the FDA regulations and is addressed in the investigator registration form FDA-1572.  All Expedited Reports must provide information about the physician to whom questions about this report are to be addressed.

 

 

Click Continue when the Reporter Information is entered completely. If any data fields are entered incorrectly, an appropriate error message will appear after Continue is clicked. Error messages will describe data problems in specific fields. Correct the problems, and click Continue again.  The data will then be saved to the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
