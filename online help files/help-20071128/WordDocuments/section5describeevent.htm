Section #5: Describe Event

 

Describe Event

 

This is one of the most critical sections of the report.  Detailed information must be provided to evaluate the adverse events and/or death reported.  Complete all of the required fields.

♦    Description

o  Provide descriptive information on the presentation of event, clinical findings, the treatment of the events, and the timing of events related to agent administration or investigational intervention

♦    Present status

o  Provide information on the current status of the patient by selecting the appropriate value from the drop down box.  For example, if the AE is currently affecting the patient, select ‘Intervention for AE Continues’

♦    Date of recovery or death

o  This field is mandatory if one of the following was selected for “Present status”: Dead, Recovered with sequelae, or Recovered without sequelae. Enter the date of death or recovery.

♦    Has the  participant been re-treated?

o  Has the intervention been administered to the patient after the Adverse Event described in this report (as of the date of this report)?

♦    Date removed from protocol

o  Enter the date in the text field or select it by using the calendar widget to the right of the field.

 

 

Click Continue when the Describe Event tab is entered completely. If any data fields are entered incorrectly, an error message will appear after Continue is clicked. Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  The data will be saved to the database

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
