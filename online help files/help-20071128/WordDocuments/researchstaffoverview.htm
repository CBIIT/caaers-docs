Research Staff Overview

Research staff needs to be created for each organization in caAERS.  While creating the staff, you can provide them login credentials by selecting user privileges at the bottom of the screen.

 

Each user role will provide different access privileges to the caAERS application.  Once the required information is entered and the organization is selected you will want to select the user role.  The user can be assigned multiple roles but the roles themselves allow the following access:

♦    Participant coordinator – This access privilege will allow the user to navigate to the Adverse Events, Studies, and Participants modules.  The user will be allowed to report AEs, create studies and create participants.

♦    Study coordinator – I have not been able to log in as a Study coordinator.

♦    Adverse event coordinator – This access privilege will allow the user to navigate to the Adverse Events module.  The user will be allowed to report AEs.

♦    Site coordinator – This access privilege will allow the user to navigate to the Adverse Events, Studies, Rules, and Administration modules.  The user will be allowed to report AEs, create studies, set up rules and have access to administrative features of the application.

The default password for the new user will be their last name and the user name will be the email address you supply.  Both the user name and password are case sensitive.  Currently, there is no way to reset a user’s password.

 

 

 

Click Save when the Research Staff section is entered completely.  The data will then be saved to the database.
