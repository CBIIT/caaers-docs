Section #2: Rule Set

A Ruleset is the group of conditions set to determine what action to take when a user enters certain criteria in a routine adverse event.  If a ruleset already exists for the organization/study you selected in the previous section, you modify it by selecting its name from the drop down box.  Otherwise, select the Create RuleSet button.

 

RuleSet Name       

♦    Select the ruleset name from the dropdown box.

o  SAE Reporting Rules determine when an expedited report must be submitted.

o  Mandatory Sections Rules determine when certain sections of the expedited report become mandatory.

 

Click Continue when the Rule Set section is completed.  If any data fields are entered incorrectly, an error message will appear after Continue is clicked.  Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  This will save the data in the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
