Adverse Events Module > Enter Expedited Report

Section #2: Enter AEs

Enter all AEs that should appear on this expedited report. For each AE, complete all the required fields, and then click Continue.

 

Enter AEs     

♦    CTC Category (Unless MedDRA is used)

o  Select the CTC category from the drop-down list.  This will limit the number of returns you will receive in the CTC Term search field.

♦    CTC Term or MeDRA Code

o  In the text field enter the first few characters for the term or code.  Based on the terminology used by the study, a drop-down list will be generated for you to select from.

o  If you select a term or code that specifies the word “Other,” another field will appear that you will need to complete.

♦    Grade

o  Enter the severity of the adverse event.

♦    Start date

o  Enter the date the adverse event began.

♦    End Date

o  Enter the date the adverse event ended

♦    Attribution to study

o  Select from the list the most appropriate term describing the relationship of the event to the investigational agent.

♦    Hospitalization

o  Report any medical event that precipitated hospitalization (or prolongation of existing hospitalization), regardless of requirements for Phase of trial, expected or unexpected, or attribution.

♦    Expected

o  If known, specify whether the AE is expected or not, as determined by the protocol guidelines. If this is a CTEP Sponsored trial, you may refer also to the AdEERS Agent Specific Adverse Event List (ASAEL).

♦    Comments

o  You may optionally provide additional information associated with the adverse event.

 

 

Click Continue when the Enter AEs section is completed.  If any data fields are entered incorrectly, an error message will appear after Continue is clicked.  Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  This will save the data in the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
