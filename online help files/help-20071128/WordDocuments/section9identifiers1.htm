Study Module > Create Study

Section #9: Identifiers

Document the identifiers for the study and whether or not they are being assigned by the organization or if the system is going to provide the identifier.  Each organization taking part in the study must assign an identifier to the study so it can be recognized at the institution.

Add as many identifiers as needed, but if you need to remove one you can select the Delete button.  Enter the information in the following fields.

♦    Identifier

♦    Identifier Type

♦    Organization Name

♦    Primary Indicator

 

Click Continue when the Identifiers tab is entered completely. If any data fields are entered incorrectly, an error message will appear after Continue is clicked. Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  The data will be saved to the database

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
