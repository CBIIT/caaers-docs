Section #4: Treatment Assignments

Treatment Assignments are used to track and analyze participants as they go through the study.  The section will appear blank until the Add Treatment Assignment button is selected in the lower right.  This will display a form for you to enter in the appropriate information for the treatment assignment. 

 

More than one treatment assignment can be added by continuing to select the Add Treatment Assignment button, and if you want to remove a treatment assignment select the red X button.

 

Treatment Assignment

♦    Code

o  Enter the code for the treatment assignment

♦    Dose Level Order

o  The amount of the dose to be given to the participants in the study

♦    Description

o  A description of what the treatment assignment is.

♦    Comments

o  Any additional comments that can be provided to define what the treatment assignments are.

 

Click Continue when the Treatment Assignment section is entered completely. If any data fields are entered incorrectly, an appropriate error message will appear after Continue is clicked. Error messages will describe data problems in specific fields. Correct the problems, and click Continue again.  The data will then be saved to the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
