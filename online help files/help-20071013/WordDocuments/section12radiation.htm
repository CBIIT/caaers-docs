Section #12: Radiation

Radiation can be us of the interventions for an AE during a study.  If Radiation information is necessary to treat this AE, select the “Add radiation” button and a form will display information to help describe the intervention that took place.

 

Treatment arm and Treatment arm description will be displayed based on previous information provided in the expedited report.

 

♦    Type of radiation administration

o  Select the radiation that was administered to the participant in the drop down box.

♦    Dosage

o  Enter the amount of radiation the participant was subjected to.

♦    Dosage unit

o  Enter the unit of measure for the amount of radiation.

♦    Date of last treatment

o  Enter the date of the last radiation administration or select the date using the calendar widget.

♦    Number of elapsed days

o  Provide the number of days that have passed since the participant underwent radiation

 

Click Continue when the Radiation tab is entered completely or if there was no radiation administered.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
