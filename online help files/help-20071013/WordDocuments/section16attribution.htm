Section #16: Attribution

For each AE, the reporter/physician must assign at least one attribution of Unrelated, Unlikely, Possible, Probable, or Definite to the Disease, Course, Surgery, Radiation, Medical device, Concomitant Medications, or Other causes.  If the sections mentioned above have not been supplied with information a message will state “No (disease/course/surgery/etc.) for this report”.

 

Agent and disease information is mandatory in Expedited Reports. Therefore agent and disease information must be entered, saved and addressed in the “Attribution for Adverse Event” section before a report can be submitted. “Concomitant Medications” and “Other Causes” are not applicable for all patients. If Concomitant Medications and Other Causes were not entered for this report, only the Protocol Agents and Disease information will appear on the table and require assignment of attribution. However, if Concomitant Medications and Other Causes were entered for this report, then Protocol Agents, Disease, Concomitant Medications and Other Causes will appear on the table and require assignment of attribution

 

Repeat the steps for the Second Event and all subsequent events if applicable. Click Continue when the Attribution section is completely filled. If any data fields are entered incorrectly, an error message will appear after Continue is clicked. Error messages will describe any problems with the data.  Correct the problem(s) and select Continue again.  The date will be saved to the database.

Click Reset tab to clear all form fields.

Click the Back button to return to the previous page.  Data entered will be maintained in the web page memory, but not be saved to the database.
