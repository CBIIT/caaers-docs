Enter Expedited Report Overview

The caAERS application allows a user to report Serious Adverse Events via a web based interface.  Once the report is entered it will be submitted based on a schedule set up according to Sponsor, Institution, or Study requirements.  There are 18 screens of information that can be captured before the report is submitted, but not every screen will have required information.

After you have selected the participant and study, there will be a box on the left of the screen with a summary of information.  This section of the page is called Summary Box and contains non-editable text listing the participant, study, report creation date/time, Primary AE, Adverse event count, ticket number, and next report due.

Efficient completion of the expedited  report is accomplished by working on the sections in order as they appear on the menu.

Depending on the study, different fields could be mandatory or optional.  Every field in the online help that references a required field means it is required in all circumstances.  A field could be mandatory for you as you go through the pages, but it might not be mandatory for another user in another study.  Information for the field will be provided.
